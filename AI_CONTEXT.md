# vite-backbone-ts — AI Context

## Overview

Minimal Backbone.js starter template powered by Vite.

## Architecture

- Client-side SPA
- Backbone Router for handling navigation
- Backbone Views for rendering UI
- Handlebars templates for view-layer markup generation
- Bootstrap 5 for styling
- jQuery + Underscore as Backbone dependencies
- Vite for bundling and dev server
- Vitest for unit tests
- Oxlint for code quality

## Entry Points

- src/main.ts
- src/main-cdn.ts

## File Organization

- collections/ → Backbone collections
- models/ → Backbone models
- views/ → Backbone views
- routers/ → Backbone routers
- styles/ → SCSS styling
- data/ → Static data

## Goals

- Lightweight
- Educational
- Extendable
- Minimal abstraction

## Non-Goals

- No SSR
- No backend
- No state management frameworks
- No heavy UI libraries
